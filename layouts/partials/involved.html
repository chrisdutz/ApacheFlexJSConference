Get Involved

Apache FlexJS conference is a great opportunity to present what you have been working on to people involved in the project.

All talks should be submitted to the ApacheCon North America (2017) event with an Apache FlexJS note.

Please specify Apache FlexJS or FlexJS in the Audience section so the talk is added to the FlexJS tracks.

Each accepted talk includes one free registration to the conference.

The Call For Participation (CFP) closes on February 11th, so get your talk in today.
